Decaf Bytecode VM

Decaf is a compact language compiler and virtual machine built entirely in Python. It includes a hand-written lexer, Pratt parser, semantic resolver, and a bytecode compiler targeting a lightweight stack-based VM. The toolchain supports lexical scoping, immutability rules, function calls, disassembly tooling, and a CLI for compiling, executing, and tracing bytecode programs.
